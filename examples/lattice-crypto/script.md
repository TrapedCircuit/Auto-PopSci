### S03 LatticeBasicsScene (1:30 - 3:15)

什么是格？其实非常直观。想象一张无限延伸的方格纸，上面的每个交叉点就构成了一个最简单的二维格。

[ANIM: 二维方格点阵出现]

更正式地说，给定n个线性无关的基向量，格就是它们所有整数线性组合构成的点集。

[ANIM: 两个基向量 → 整数组合生成格点]

关键在于，同一个格可以由不同的基来表示。有的基向量近似正交，排列整齐，我们称之为"好基"。有的基向量几乎平行、极度倾斜，这就是"坏基"。

[ANIM: 同一格，好基 vs 坏基的对比动画]

好基让格上的计算变得容易，而坏基则让同样的问题变得极其困难。这种好基与坏基之间的不对称性，正是格密码学安全性的根基。

[ANIM: 好基上轻松找到最短向量 → 坏基上完全迷失]

如果有人给你一组坏基，想要从中恢复出好基，在高维空间中，这几乎是不可能的任务。

[ANIM: 高维空间抽象表示 → 维度爆炸效果]

### S04 HardProblemsScene (3:15 - 4:45)

格密码学的安全性建立在两个核心困难问题之上。第一个叫最短向量问题，简称SVP。

[ANIM: 格点中标注原点 → 高亮最短非零向量]

给定一个格的基，找到格中距离原点最近的非零向量。在低维空间中这看起来不难，但当维度上升到几百维甚至上千维时，搜索空间呈指数级增长，目前没有任何已知的算法——包括量子算法——能够高效解决它。

[ANIM: 从2D到3D到高维的维度提升动画 → 搜索空间指数爆炸]

第二个核心问题叫最近向量问题，简称CVP。给定一个目标点，找到格中距离它最近的格点。

[ANIM: 目标点出现 → 搜索最近格点的过程]

CVP比SVP更难，它是NP困难问题。这两个问题的计算困难性，为格密码学提供了坚不可摧的数学基础。

[ANIM: SVP和CVP并列展示 → "抗量子"标签出现]

### S05 LWEScene (4:45 - 6:45)

现在让我们进入格密码学最核心的构造——带错误学习问题，英文简称LWE。这是格密码学从理论走向实用的关键一步。

[ANIM: LWE标题出现]

LWE的设定是这样的。假设有一个秘密向量s，我们不断选取随机向量a，计算它和s的内积，然后加上一个很小的随机误差e。这样就得到了一组近似等式。

[ANIM: s向量显示 → a·s + e 的计算过程动画]

直觉上，如果没有误差e，这就是一个简单的线性方程组，用高斯消元法就能轻松求解。但加上微小的噪声后，问题的性质发生了根本性的变化。

[ANIM: 精确方程 → 加入噪声 → 方程变得模糊]

为什么噪声让问题变得如此困难？因为噪声破坏了线性结构。你无法确定每个等式的偏差有多大，也就无法通过联立方程来消除未知数。即使只有很小的误差，在高维空间中，这些误差会积累和放大，让任何求解尝试都偏离正确答案。

[ANIM: 噪声在方程组中传播的可视化]

Oded Regev在2005年证明了一个里程碑式的定理：解决LWE问题和解决格上的SVP问题一样困难。这意味着，如果你能破解LWE，就等于你也能解决格上最困难的问题。LWE的安全性直接扎根于格的计算困难性。

[ANIM: LWE ↔ SVP 的归约关系图]

### S06 EncryptionScene (6:45 - 8:15)

那么如何用LWE来构造一个真正的加密方案呢？让我们来看看基本思路。

[ANIM: 加密方案标题]

Alice作为接收方，首先生成密钥。她随机选择一个秘密向量s作为私钥。然后生成多组随机向量a和对应的近似内积b等于a点乘s加噪声e。公钥就是这些a和b的集合。

[ANIM: Alice生成私钥s → 计算公钥(A,b)]

现在Bob要发送一个比特消息m给Alice。他从公钥中随机选取若干行进行求和，得到一个聚合向量u和聚合值v。如果m等于1，就再给v加上q除以2。

[ANIM: Bob选取行 → 求和 → 加密过程]

Alice收到密文后，用私钥s计算v减去u点乘s。由于噪声很小，结果要么接近0，要么接近q除以2，正好对应消息0或1。

[ANIM: Alice解密过程 → 正确恢复消息m]

这就是LWE加密的精妙之处：加入噪声让攻击者无法破解，但噪声又足够小，使得合法接收者能够正确解密。

[ANIM: 安全性 vs 正确性 的平衡天平]

### S07 ApplicationsScene (8:15 - 9:30)

格密码学不仅是理论上的突破，它正在改变整个密码学的格局。

[ANIM: 应用场景展示]

2022年，美国国家标准与技术研究院NIST正式选定了首批后量子密码标准。其中，密钥封装方案CRYSTALS-Kyber和数字签名方案CRYSTALS-Dilithium，都基于格密码学。从浏览器到手机，从云服务到政府通信，格密码学正在全面部署。

[ANIM: NIST标准 → Kyber/Dilithium卡片 → 部署场景]

更令人兴奋的是，格密码学还催生了全同态加密技术。这种技术允许你在完全不解密的情况下对密文进行任意计算。想象一下，云服务器可以处理你的数据，却完全不知道数据内容。这在医疗、金融、隐私计算等领域有着巨大的应用潜力。

[ANIM: FHE概念图——密文上的加法和乘法]

格还是构造先进零知识证明的利器，为区块链和数字身份提供后量子安全保障。
